# GregoLy
Python script that converts Gregorian scores to Modern notation

# Roadmap
- [ ] Produce PDFs from different rhythmic schools:
  - [ ] Proportional Rhythm from [Cantatorium](https://cantatorium.com)
    - [ ] Model
    - [ ] Parser
    - [ ] Formatter
  - [ ] Classical Solesmes Rhythm from [GregoBase](https://gregobase.selapa.net)
    - [ ] Model
    - [ ] Parser
    - [ ] Formatter
  - [ ] Vatican Edition Rhythm [GregoBase](https://gregobase.selapa.net)
- [ ] Add ability to customize output
  - [ ] Key
  - [ ] How to notate special neumes (quilisma, orsicus, etc.)
  - [ ] Looks (staff color, size, etc.)
- [ ] Create demo
- [ ] Create web frontent
